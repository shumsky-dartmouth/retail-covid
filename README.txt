This directory contains code for the paper "Retail Store Customer Flow and COVID-19 Transmission."

Files that begin "Figure..." are maple files that generate each of the figures in the paper.

The file "run_gsim.m" runs instances of the discrete event simulation and calls the function "gsim.m". The files "samec.mat" and "superspreader.mat" are matlab data files generated by the simulation runs called by "run_gsim.m".
